# frontend

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Run your unit tests

[Jest](https://jestjs.io/) is used for unit tests.

Run tests with:

```
yarn test:unit
```
